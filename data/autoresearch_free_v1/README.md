# AutoResearch Free V1

Random free-response splits for LoRA hyperparameter search.

Generated by:

```bash
python scripts/build_autoresearch_free_splits.py \
  --input data/hf_mixed_math_free_100k/train.jsonl \
  --out-dir data/autoresearch_free_v1 \
  --seed 151 \
  --train-size 5000 \
  --dev-size 1000 \
  --holdout-size 1000 \
  --max-reasoning-words 700 \
  --no-oracle-score-filter
```

## Files

- `train.jsonl`: 5,000 free-response SFT rows for first-stage hyperparameter search.
- `train_parts/train-0000.jsonl`: same rows as `train.jsonl`, kept for consistency with larger split layouts.
- `dev_sft.jsonl`: 1,000 held-out SFT rows for token-loss sanity checks only.
- `holdout_sft.jsonl`: 1,000 held-out SFT rows. Do not use for hyperparameter search.
- `dev_benchmark.jsonl`: 1,000 held-out benchmark-format rows for generated-answer dev scoring.
- `holdout_benchmark.jsonl`: 1,000 held-out benchmark-format rows for final confirmation scoring.
- `manifest.json`: split metadata.

## Filtering

Rows are free-response only and randomly sampled after filtering out:

- MCQ-like prompts.
- Missing or malformed `FINAL_ANSWERS:\n\boxed{...}` targets.
- Letter-only targets.
- Reasoning traces longer than 700 words.
- Answer shapes that are unstable under the local judge, including `\text{...}`, units, labels, commas, currency, percentages, and `\dfrac`.

## Benchmark Sanity Check

Fresh oracle benchmark scoring was run with gold answers as predictions:

- `dev_benchmark.jsonl`: 1000/1000 = 100.00%
- `holdout_benchmark.jsonl`: 1000/1000 = 100.00%

The oracle results are recorded in `results/autoresearch_free_v1/summary.csv`.

The 5,000-row training split was also audited with the same local judge extraction path:

- flagged answer-format patterns: 0
- extraction failures: 0
- oracle scoring failures: 0

## Intended Use

Use `train.jsonl` for AutoResearch hyperparameter search. After selecting the best simple configuration by generated dev accuracy and confirming once on holdout, retrain that configuration on the full free-response dataset in `data/hf_mixed_math_free_100k/`.
