# HF Mixed Math 100k

Generated by:

```bash
python scripts/build_hf_mixed_math_sft.py --out-dir data/hf_mixed_math_100k --seed 151
```

## Contents

- `train.jsonl`: 100,000 chat-format SFT rows with reasoning traces.
- `train_parts/*.jsonl`: the same 100,000 rows split into four 25k-row chunks for GitHub file-size compatibility.
- `manifest.json`: generation seed, requested counts, and realized source counts.

## Source Mix

- 60,000 rows from `AI-MO/NuminaMath-CoT`
- 12,000 rows from `EleutherAI/hendrycks_math`
- 18,000 rows from `allenai/math_qa`
- 10,000 rows from `microsoft/orca-math-word-problems-200k`

## Format

Each row has:

- `id`
- `source`
- `metadata`
- `messages`: system/user/assistant chat messages

The assistant message keeps the source reasoning trace. When a clean final answer can be extracted, the assistant response ends with:

```text
FINAL_ANSWERS:
\boxed{answer}
```

In this generated file, 91,440 rows have a normalized final-answer box. The remaining rows still contain the source reasoning/solution trace, but no reliably extracted final answer.

## Suggested Use

Use this as external reasoning pretraining or mixed SFT data. For final benchmark-format tuning, follow with the local public SFT dataset so the model re-learns the exact `[ANS]` and `FINAL_ANSWERS` formatting.
