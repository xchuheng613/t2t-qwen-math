# AutoResearch Free Full Clean

Clean full free-response training set for the final LoRA retrain after hyperparameter selection.

Generated by:

```bash
python scripts/build_autoresearch_free_splits.py \
  --input data/hf_mixed_math_free_100k/train.jsonl \
  --out-dir data/autoresearch_free_full_clean \
  --seed 151 \
  --train-size 0 \
  --dev-size 0 \
  --holdout-size 0 \
  --use-all-train \
  --max-reasoning-words 700 \
  --no-oracle-score-filter
```

## Files

- `train.jsonl`: 42,402 clean free-response SFT rows.
- `train_parts/train-0000.jsonl`: first 25,000 rows.
- `train_parts/train-0001.jsonl`: remaining 17,402 rows.
- `manifest.json`: generation metadata.

The empty `dev_*` and `holdout_*` files are placeholders from the shared split builder. Use the fixed evaluation files in `data/autoresearch_free_v1/` for dev/holdout.

## Audit

The full clean training split was audited with the local judge extraction/scoring path:

- rows: 42,402
- bad schemas: 0
- flagged answer-format patterns: 0
- extraction failures: 0
- oracle scoring failures: 0

Filtered unstable answer shapes include:

- MCQ-like prompts and letter-only targets.
- Missing or malformed `FINAL_ANSWERS:\n\boxed{...}` targets.
- Reasoning traces longer than 700 words.
- `\text`, `\mathrm`, `\mathbf`, `\mbox`, `\dfrac`, `\tfrac`.
- Commas, full-width commas, currency, percentages, labels/colons, equations, units, and selected Unicode relation/logical symbols.

## Intended Use

Use this only after the 5,000-row AutoResearch search identifies a good simple configuration. Do not use this full clean set for hyperparameter search.
